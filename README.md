2. Цель работы
Разработка программы для проверки и вычисления математических выражений с графическим интерфейсом.

3. Постановка задач
   
1)Принимает на вход строку с математическим выражением
2)Проверка корректности выражения
3)Вычисление результата
4)Обрабатываение ошибок (деление на ноль, неверные символы, скобки)
5)Имеет графический интерфейс

4. Описание программы
   
4.1. Алгоритм работы:
1)Проверка символов на допустимость
2)Вычисление выражения
3)Обработка исключений

4.2. Обрабатываемые ошибки:
1)Недопустимые символы
2)Деление на ноль
3)Синтаксические ошибки

5. Тестирование
Пользователь вводит выражение в текстовое поле и нажимает кнопку запуска
2+2 = 4 (4), 2x3 = 6 (6), 2+2x3 = 8 (8).

6. Вывод
 Разработанная программа:

Корректно вычисляет математические выражения

Обнаруживает и сообщает об ошибках

Безопасна при использовании
